# rootnative

Shorthand package for [`@rootnative/cli`](https://github.com/rootnative/ui/tree/main/packages/cli).

```bash
npx rootnative create my-app
npx rootnative init
npx rootnative add button card
```

This package delegates all commands to `@rootnative/cli`. See the [CLI docs](https://rootnative.github.io/ui/cli) for the full command reference.
